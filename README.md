# 欢迎

**If you are a non-Chinese user, I recommend you to use Google Translate or deepl to translate this document.**

## ✨ 特性

* 图片懒加载
* 内置多语言
* 支持多款评论插件
* 网页访问统计
* 侧栏备案信息
* 灯箱大图
* 图片瀑布流
* 社交按钮
